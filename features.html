Core Features

The full stack of capabilities that make Hecos a sovereign AI Operating Layer.

🪐

Core Modules

The system kernel is built on Core Modules. These form the essential foundation for memory management, configuration persistence, and inter-process communication between AI layers.

🧩

Plugin Ecosystem

Logic is modular. Enable or disable specialized plugins for file management, web search, media control, or reminders on the fly without rebooting the system core.

🛠️

Extensions & Widgets

Power-user tools for deep desktop integration. Extensions allow Hecos to interact with external software, system environments, and the modular Control Room widget ecosystem.

🌐

MCP Native Bridge

Native Model Context Protocol support. Connect to any MCP Server to instantly grant the AI access to remote databases, APIs, and standardized toolkits.

🤖

Autonomous Agentic Loop

Watch Hecos reason in real-time. By utilizing multi-step Chain-of-Thought, the AI breaks down complex missions, self-corrects, and executes until completion.

🛡️

AST-Security Jail

A sophisticated Abstract Syntax Tree analysis layer filters all AI-generated code. This creates a secure "Jail" where Python can solve analytical problems without compromising the host OS integrity.

💾

Semantic Memory

Utilizes SQLite to store not just chat history, but context and learned facts about the user. The AI evolves with you over time, building a persistent cognitive profile.

🎙️

Voice Interaction

Low latency Speech-to-Text and high-quality local Text-to-Speech (Piper TTS with streaming) enable a natural, hands-free experience with minimal latency.

🖥️

OS Automation

Control mouse, keyboard, windows, files, and applications natively. Hecos can act as your personal OS shell enhanced with AI-level awareness and decision-making.

🌍

Browser Automation

Interact semantically with web pages via a managed Chromium instance. Navigate, click, extract data, and automate browser-based workflows with natural language instructions.

📡

Control Room & Widgets

A dedicated masonry-style dashboard within the WebUI where multiple interactive widgets run simultaneously — OS telemetry, weather, calendar, and more — in real-time iframes.

🔀

Local & Cloud Fluidity

Works entirely offline with Ollama or KoboldCpp on your hardware, or integrates seamlessly with OpenAI, Anthropic, Google, and any LiteLLM-compatible cloud provider.
