⚡ Core Features
v0.26.0

The full stack of capabilities that make Hecos a sovereign AI Operating Layer.

🪐
Core Modules

The system kernel is built on Core Modules. These form the essential foundation for memory management, configuration persistence, and inter-process communication between AI layers.

📦
Hecos Package Manager (HPM)

The definitive 8-layer architecture. From Core Modules and Plugins to standalone Apps, Widgets, Themes, and Skill Packs. Expand Hecos instantly by dropping signed .hpkg bundles into the WebUI, ensuring absolute code isolation.

🛠️
Extensions & Widgets

Power-user tools for deep desktop integration. Extensions allow Hecos to interact with external software, system environments, and the modular Control Room widget ecosystem.

🌐
MCP Native Bridge

Native Model Context Protocol support. Connect to any MCP Server to instantly grant the AI access to remote databases, APIs, and standardized toolkits.

🤖
Autonomous Agentic Loop

Watch Hecos reason in real-time. By utilizing multi-step Chain-of-Thought, the AI breaks down complex missions, self-corrects, and executes until completion.

Flows Automation Engine

Complete visual drag-and-drop node editor for orchestrating complex, multi-step triggers and actions. Integrated tightly with the NLP engine, allowing you to instantly generate robust YAML workflows through natural voice commands.

🛡️
AST-Security Jail

A sophisticated Abstract Syntax Tree analysis layer filters all AI-generated code, creating a secure "Jail" where Python can solve analytical problems without compromising the host OS.

💾
Semantic Memory

Utilizes SQLite to store not just chat history, but context and learned facts about the user. The AI evolves with you, building a persistent cognitive profile over time.

🔐
Centralized Backup & Recovery

Your digital sovereignty, secured. A one-click backup mechanism that instantly archives your entire memory vault, active configurations, custom flows, and downloaded packages into a single, restorable snapshot.

🎙️
Voice Interaction

Low latency Speech-to-Text and high-quality local Text-to-Speech (Piper TTS with real-time streaming) enable a natural, hands-free experience with minimal latency.

🎭
Native Personas

Hot-swap the "soul" of the machine. Switch from a strict, logical system core to a friendly companion or a formal British butler. Each persona has its own system prompt, avatar, and distinct behavioral traits.

🖥️
OS Automation

Control mouse, keyboard, windows, files, and applications natively. Hecos acts as your personal OS shell enhanced with AI-level awareness and decision-making.

🌍
Browser Automation

Interact semantically with web pages via a managed Chromium instance. Navigate, click, extract data, and automate browser-based workflows.

📧
Mail Orchestration

Native SMTP/IMAP integration. Send and receive emails through AI-driven workflows with intelligent contact lookup and draft generation.

🎥
Live Vision Feed

Real-time webcam and system vision integration in the Control Room. Monitor local feeds or analyze desktop screenshots with low-latency projection.

⚡
Direct Commands

Bypass the LLM entirely for critical, time-sensitive tasks. Use exact keyword triggers to instantly mute audio, lock the screen, or launch specific flows with zero latency and 100% reliability, even completely offline.

📋
Lists & Task Management

A highly structured data store for your daily life. Manage dynamic checklists, project milestones, and shopping inventories directly from the chat. The AI autonomously checks off items and triggers events when lists are completed.

📅
Calendar Integration

Native calendar management with event creation, scheduling, and retrieval via voice or automation. Syncs with the Flows engine for time-based trigger workflows.

🔔
Reminders & Notifications

Set time-based and context-aware reminders by voice or automation. Hecos handles the scheduling and delivers alerts through audio, TTS, and the Control Room dashboard.
